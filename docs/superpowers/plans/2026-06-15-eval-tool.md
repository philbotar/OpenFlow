# Eval Tool Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an OMP-style `eval` builtin with persistent Python and JavaScript cells, per-cell timeout/reset, `display()` capture, and live `ToolUpdated` progress.

**Architecture:** Own cell runtimes in `orchestration/src/eval/`. Python: long-lived subprocess with JSON-lines RPC (not full Jupyter in v1). JavaScript: persistent `deno eval --watch` or `node --eval` subprocess with same RPC envelope. One `EvalSession` per run on `ToolRunner`, keyed by language. Exclusive concurrency. Cells execute in order; state persists across tool calls until `reset: true`.

**Tech Stack:** Rust, Tokio subprocess, serde_json, existing `ArtifactStore`, `ToolUpdated`.

**Reference:** `oh-my-pi/docs/tools/eval.md`, `oh-my-pi/docs/python-repl.md`.

## V1 Scope

- Input: `{ "cells": [ { "language": "py"|"js", "code": "...", "title"?, "timeout"?, "reset"? } ] }`
- Min 1 cell; max 20 cells per call.
- Timeout per cell: default 30s, clamp 1–600s.
- `reset: true` wipes that language's session only.
- Output: combined text + structured per-cell status in tool result metadata.
- `display(value)` appends to cell output (JSON pretty-print objects).
- Image display deferred (text/JSON only in v1).
- Require `python3` and `node` on PATH; clear error if missing.

## Out of Scope

- IPython/Jupyter gateway (v2).
- `agent()` bridge into subagents.
- Markdown rendering in UI (text bubble only).
- Bun-specific runtime.

    pub title: Option<String>,
    #[serde(default)]
    pub timeout: Option<u64>,

#[serde(rename_all = "lowercase")]
pub enum EvalLanguage {

    Js,
}

#[derive(Debug, Serialize, Deserialize)]
pub struct EvalCellResult {
    pub title: Option<String>,
    pub status: EvalCellStatus,
    pub output: String,
    pub duration_ms: u64,
}

#[derive(Debug, Serialize, Deserialize)]
#[serde(rename_all = "lowercase")]
pub enum EvalCellStatus {
    Complete,
    Error,
}
